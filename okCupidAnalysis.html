Analysis of OkCupid Dating Profiles¶

Introduction¶

Context¶

OkCupid is a free online dating app where users have profiles that must have at least one picture and could possibly have some prompt questions that ask about their life style. The data was scraped off the app with permission from app co-founders.

Dataset¶

The data set itself found on github: https://github.com/rudeboybert/JSE_OkCupid?tab=readme-ov-file. The data is spread across two seperate csv files one called profiles_revised.csv.zip and essays_revised_and_shuffled.zip. The profiles_revised.csv contains profile data for 59,946 people in San Francisco. This file contains mostly the lifestyle and demographics details about the users. The fields along with descriptions are as follows:

age: age of user with random noise added for anonymization

body_type- rather not say, thin, overweight, skinny, average, fit, athletic, jacked, a little extra, curvy, full figured, used up

diet- mostly/strictly; anything, vegetarian, vegan, kosher, halal, other

drinks- very often, often, socially, rarely, desperately, not at all

drugs- never, sometimes, often

education- graduated from, working on, dropped out of; high school, two-year college, university, masters program, law school, med school, Ph.D program, space camp

ethnicity- Asian, middle eastern, black, native American, indian, pacific islander, Hispanic/latin, white, other

Data Cleaning¶

The essay portion of the data set was anonymized by making not matching it up with the profile data from the first csv file. So row 1 in the profiles csv does not mean row 1 in the essay portion to allow for the privacy of people. Also a noise factor was added to ages to once again maintain people's privacy. Also any links to the individuals social media accoutns such as facebook, instagram, twitter, etc. were removed.

def clean_text(text):
    text = re.sub(r"http\S+", "", text)  # remove URLs
    text = re.sub(r"\S+@\S+", "", text)  # remove email addresses
    text = re.sub(r"<.*?>", "", text)  # remove HTML tags
    text = re.sub(r"[^A-Za-z\s]", "", text.lower())  # remove non-alphabets
    tokens = word_tokenize(text)  # tokenize
    tokens = [word for word in tokens if word not in stop_words]  # remove stopwords
    return " ".join(tokens)

# Check if the function works for a sample text

print("Cleaned sample text:", clean_text(sample_text))

# Apply the cleaning function to each essay
for col in essays_df.columns:
    essays_df[col] = essays_df[col].apply(lambda x: clean_text(str(x)))

# Display the first few rows after cleaning

# Combine all essays into a single text for each user

def sentences_to_words(sentences):
    for sentence in sentences:
        yield(simple_preprocess(str(sentence), deacc=True))

def remove_stopwords(texts):
    return [[word for word in simple_preprocess(str(doc)) if word not in stop_words] for doc in texts]

essays_text = essays_df['all_essays'].tolist()

# Create Dictionary
